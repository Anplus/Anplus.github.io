2017 start again！

半年过去了，断断续续写了一点东西，但是总没机会整理到博客上。时间真的过的好快，转眼已经是五月份了。总觉得未经整理的生活少点什么，但在别的地方写点东西又觉得有些浮躁，想来想去还是回到这里。

知易行难，一言难尽，那就开始吧。

最后用一句话作结。

转载自豆瓣

曾经充满热情，仿佛知道的努力的方向，却被渐渐笼罩心头的无力感带入无所事事的混沌，终将耗尽那宝贵的热情。重要的问题不是不知道向上的途径，而是明明知道自己可以做的更好，却没有一个长久激励自己，保持忘我的热情的源泉。
