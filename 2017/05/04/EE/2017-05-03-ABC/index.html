Ambient Backscatter Communication

发表于 2017-05-04   |   分类于 Signal Processing

Sigcomm2013的best paper是来自华盛顿大学wisp组关于Ambient Backscatter Communication(环境反向散射波通信)的文章。这项工作实现了无源的传感器通信节点，令人十分震撼。

1. 整体思路

首先，传感器节点要能够收集环境散射波的能量，为微处理器芯片和传感器供电，所以需要一个电源收集模块。环境散射波的能量只有微瓦级别，勉强能够超低功耗的单片机工作。这样其他的模块尽量要做到少耗能。其次，传感器要与外界通信，一定要有收发模块。最后加之微处理器和传感器，以及一系列用户接口。

。

2. 电源模块

电源模块是wisp组之前的工作，利用的是一组肖特基二极管和电容组成的升压电路。利用二极管的单向导通，不断利用之前充好电的电容给新的电容充电实现升压。

3. 发射模块

为了节省能量，ABC不直接发射信号，而是利用反射和不反射环境散射波两种状态来传输0和1。两个靠近的节点通信的时候，反射叠加已有的环境中的无线电波和只有环境的无线电波两种状态来反应数字信号的0和1。

发射模块利用一个三极管控制天线的短路和导通。利用传输线的理论，天线短路的时候反射系数为1，全部反射环境散射波。而天线通路的时候，全部吸收环境散射波。

还有要考虑的问题是天线的设计，要能够保证通断两种状态的阻抗差别够大，显然这是和频带有关系的。这里利用的是电视塔的信号，中心频率为540Mhz，带宽50Mhz，天线选择为258mm的双极天线。

4. 接收模块

由于环境散射波是随机的，如何从这个随机的信号中恢复我们的信息是个比较难解决的问题，而且在解调的过程中还要尽量减少能量消耗。虽然信号是随机的，但是我们知道在一段时间内，发零和发一的信号累积能量不同。ABC先经过包络检波器，得到信号的包络，这个时候已经能大致区分出两种状态的信号了。进一步利用一个均值电路得到两种状态的区分阈值。最后通过比较器得到01序列。

这里面ABC选择了FM0编码，FM0编码是双相间空号编码，工作原理是在一个位窗内采用电平变化来表示逻辑,这种编码的好处是高低电平出现的概率一样。这样均值电路得到的一定是最优的阈值，而且不会由于出现长段的零导致的充电不够。

5. 载波检测

多个节点通信的时候，为了避免信道冲突，我们需要实现载波检测。实际上，由于前面通信的过程的特点，导致如果有节点之间在通信，01出现的次数接近；没有通信的时候，都为0。这样我们只需要检测一段信道，统计一下01的次数就可以判断有没有通信。

6. 问题

无源节点之间通信，如何才能保证两个节点都是充好电的。文中利用一段比较长的前导码为节点充电，但是仅靠这个能不能保证每次都充好电。如果没充好电，通信一定是失败的，但是通信双方都无法解决这个问题。
