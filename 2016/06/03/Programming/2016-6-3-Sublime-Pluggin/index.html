Sublime 插件开发尝试

发表于 2016-06-03   |   分类于 Programming

Sublime Text是我比较喜欢的文本编辑器，一直使用别人开发的插件。最近尝试了下Sublime插件的开发，写了个小例子，这里整理一下。
注意一下都是在windows+sublime Text3环境下完成的。

1. 添加自己的插件

选择菜单里面的
Tools -> Developer -> New pluggin…
就可以看到插件的Hello world例子了。

其中类名ExampleCommand中Command之前的就是之后调用的插件ID。

2. 调试插件

  1. 保存插件到 ~\Sublime Text 3\Packages\User\
  2. Ctrl+` 打开Sublime Text的控制台
  3. 运行example（这里填入插件的ID）

sublime本身就带python的编辑器，不用配置python的环境，直接运行就可以。

4. 编写插件

Sublime Text的插件都是利用python编写的，掌握一点python基础还是必要的。
这里先简要记录一下常用的几个API：

  1. View
    view是text buffer(缓冲区)中的视图。
    利用self.view放访问当前view

  2. Region(a, b)
    Region是标记缓冲区的里的一段区域的特有数据结构
    Region.a返回区域的起始位置
    Region.b返回区域的结束位置

  3. 由于是文本编辑器，肯定要涉及到众多的文本查找功能，可以使用view里面的find_all方法，支持正则表达式

  1. 判断Region为空，要用Region == []
